General information

The Zoneminder module is designed to support the Zoneminder video surveillance system using an API.

The module displays a list of cameras, their functions, and also lists events by hour, day, week, and month. Video viewing of events is available.

The module can be integrated into menus and scenes.

Zoneminder setup

The Zoneminder API is enabled by default. To check, you can go to Options->System to check the checkbox OPT_USE_API.

Attention! Zoneminder v1.36.36 has a bug that prevents retrieving the event list. An earlier version or update to v1.36.37 is required.

Module setup

To configure the module, you must specify the server address in the settings panel.

Embedding into the menu

To embed a module, use a link of the following type: [#module name="zoneminder" monitor="1" interval="2000" showeventlist="1" scale="100"#].

Available options:

Parameter Description Default
monitor Monitor Id required
interval Refresh interval in ms 1000
showeventlist Show latest events list no
scale Image quality (0-100%) 100
