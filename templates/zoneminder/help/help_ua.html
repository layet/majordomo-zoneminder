Загальні відомості

Модуль Zoneminder призначений для підтримки системи відеоспостереження Zoneminder з використанням API.

Модуль дозволяє виводити список камер, режими їх роботи, а також списки подій за годину/день/тиждень та місяць. Перегляд відео події доступний.

Модуль дозволяє вбудовувати його в меню та сцени.

Налаштування Zoneminder

API Zoneminder увімкнено за замовчуванням. Для перевірки можна зайти в Options->System та перевірити чекбокс OPT_USE_API.

Увага! В Zoneminder v1.36.36 присутня помилка, що не дозволяє отримати список подій. Потрібна рання версія або оновлення до v1.36.37.

Налаштування модуля

Для налаштування модуля необхідно вказати адресу сервера на панелі опцій.

Вбудовування у меню

Для вбудовування модуля використовується посилання виду [#module name="zoneminder" monitor="1" interval="2000" showeventlist="1" scale="100"#].

Доступні параметри:

Параметр Опис Значення за замовчуванням
monitor № камеры обов'язковий
interval Інтервал оновлення у мс 1000
showeventlist Показати список останніх подій нет
scale Якість зображення (0-100%) 100
