Общие сведения

Модуль Zoneminder предназначен для поддержки системы видеонаблюдения Zoneminder с использованием API.

Модуль позволяет выводить список камер, режимы их работы, а также списки событий за час/день/неделю и месяц. Доступен просмотр видео события.

Модуль позволяет встраивать его в меню и сцены.

Настройка Zoneminder

API Zoneminder включено по-умолчанию. Для проверки можно зайти в Options->System и проверить чекбокс OPT_USE_API.

Внимание! В Zoneminder v1.36.36 присутствует ошибка не позволяющая получить список событий. Необходима более ранняя версия или обновление до v1.36.37.

Настройка модуля

Для настройки модуля необходимо указать адрес сервера в панели настроек.

Встраивание в меню

Для встраивания модуля используется ссылка вида [#module name="zoneminder" monitor="1" interval="2000" showeventlist="1" scale="100"#].

Доступные параметры:

Параметр Описание Значение по-умолчанию
monitor № камеры обязателен
interval Интервал обновления в мс 1000
showeventlist Показать список последних событий нет
scale Качество изображения (0-100%) 100
Встраивание в сцену

Для встраивания модуля используется ссылка вида [#module name="zoneminder" view_mode="mobile" monitors="1,2,3" interval="2000" showeventlist="1" scale="100"#].

Доступные параметры:

Параметр Описание Значение по-умолчанию
view_mode Мобильный вид желателен
monitors Список камер через запятую обязателен
interval Интервал обновления в мс 1000
showeventlist Показать список последних событий нет
scale Качество изображения (0-100%) 100
